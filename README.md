# hoard

## Install

### From source

It's best to use [rustup](https://rustup.rs/) to get setup with a Rust
toolchain, then you can run:

```
cargo build --release
```

Find the binaries in `target/release/hoard`
Move it to wherever you need it


### Homebrew

```
brew tap Hypde46/hoard
brew install hoard
```
